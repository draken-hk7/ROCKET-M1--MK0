# ROCKET M1-MK0 Design Report

ROCKET M1-MK0 is a simulation-first, non-propellant aerospace research
airframe. The current design baseline focuses on:

- biomimetic geometry and Fibonacci-derived layout rules
- parametric educational CAD
- Barrowman-style external aerodynamics
- structural screening calculations
- ESP32 avionics and LoRa telemetry
- OpenFOAM external aerodynamic case setup

No combustion, propellant chemistry, pressure-vessel fabrication, ignition, or
energetic deployment content is included.

Detailed requirements are captured in:

- `docs/engineering_specification.md`
- `docs/fibonacci_design_philosophy.md`
- `docs/system_architecture.md`
- `docs/safety_considerations.md`
