# Safety Notes

This repository is for non-propellant educational airframe simulation and
avionics development only.

The repository intentionally excludes:

- combustion chemistry
- energetic propellants
- pressure-vessel fabrication specifications
- ignition systems
- hot-fire procedures
- energetic parachute deployment hardware

Use the scripts, CAD, FEA, CFD, telemetry, and firmware as educational
engineering references. They are not flight qualification or certification data.
