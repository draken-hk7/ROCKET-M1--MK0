# Hot-Fire Sequence

No hot-fire sequence is provided in this repository.

Rocket hot-fire testing requires lawful authority, certified facilities,
qualified personnel, approved procedures, emergency planning, instrumentation
review, and range safety control. This project intentionally omits ignition
design, valve timing, abort thresholds, propellant handling, and shutdown
procedures.

Use `test/test_plan.md` for non-propellant educational workflow checks only.
